AIBO THERAPY

QUESTIONS AND ANSWERS

Q: AIBO won't start up/won't start properly.
A: Make sure that you have properly installed AIBO THERAPY. Put AIBO on a flat surface and ensure that you are using a programmable memory stick. If you're still having issues, send me an email.

Q: AIBO can't hear me.
A: Ensure that you are in a quiet environment. Hold your AIBO's head still if head jitters are severe.

Q: My AIBO goes into jam mode while running AIBO therapy.
A: An AIBO in good condition will not jam while running AIBO therapy. Make sure nothing gets caught in AIBO's joints- if nothing is stuck, clean your robot's joints manually before attempting to use AIBO THERAPY.

Q: I'm pressing the head button, but AIBO won't stop.
A: Make sure you are holding down the head button until AIBO finishes it's 'cycle'. The head shutdown function is NOT emergency stop.
